# Dotfiles
Synchronize personal dotfiles, and install required packages.
## Steps
1) Download Xcode
2) Run ```link.sh``` for symlink.
3) Install Homebrew to `$HOME/.homebrew`:
   - `git clone https://github.com/brew.git $HOME/.homebrew`
   - `brew analytics off`
4) Install Homebrew packages:
   - `brew bundle --global`
